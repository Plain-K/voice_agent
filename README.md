🚀 Voice Agent（语音智能代理系统）
一个基于语音识别 + 大语言模型的智能代理系统，支持电话呼入和企业微信自动推送。

🧩 系统架构
## 电话层
通过livekit来实现电话层功能：包括将呼入者音频切割为pcm16格式，以及将tts生成的音频回拨

## 语音流层
### 使用stt-llm-tts的工作流，并没有使用实时语音api 
如何实现信息收集？ 使用task group来规定需要收集的信息，并让大模型调用 收集完毕的信息会返回到_finish_data_collection中，并在此实现企业微信推送

## 微信层
通过企业微信的webhook将预约信息发送到企业微信群
